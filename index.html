Welcome to The Fan-Made Heroes II Monster Database!

This is a hobby project for a dear game that originally came out in 1998 and I have been playing it since 2000. Heroes II is the second game in the franchise and (arguably for some but not me) THE BEST one so far and (probably) forever. If you know the game, props to you!
If you know the franchise but not the game, give it a shot!

Really enjoyed building this one and the JSON file of it. I plan on visiting this lovely project time to time to add new things and improve it. As it stands, however, enjoy it!

-Raven
